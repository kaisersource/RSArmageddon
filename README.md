<ul>
    <li>Debian 10
    <li>Kali linux 2020.2
    <li>Arch linux
</ul>

<p>work in progress for Windows</p>

## How to install (Kali linux, Debian)

```
$ git clone https://github.com/m1gnus/RSArmageddon.git && cd RSArmageddon && ./setup_deb.sh
...
$ rsarmageddon
```
